Real-time cryptocurrency price tracker with live charts, search, favorites, and dark/light mode.

Features
Live price updates for top 100 coins
Interactive historical charts (1D/7D/30D/1Y)
Coin search and filter
Responsive mobile-first design
Dark/light theme toggle
Local favorites/watchlist
Tech Stack
Vanilla JavaScript, CoinGecko API, Chart.js, Tailwind CSS, localStorage

Quick Start
Clone repo: git clone https://github.com/VIC-gt/cryptotracker2.git
cd cryptotracker2
npm install && npm start
Live Demo
[https://vic-gt.github.io/cryptotracker2/]

Usage
Browse top coins by market cap
Search any coin by name/symbol
Click coin for detailed chart
Star coins to add to favorites
Toggle light/dark theme
API
CoinGecko Free API - Live prices, historical charts, search (10-30 calls/min)

Project Structure
index.html - Main page
css/ - Stylesheets
js/ - main.js, api.js, chart.js, storage.js
assets/ - Icons and images
Deployment
Netlify, Vercel, or GitHub Pages - Just drag and drop!

Customization
Edit config.js for currency, themes, chart periods, API delays

Browser Support
Chrome 90+, Firefox 85+, Safari 14+, Edge 90+
